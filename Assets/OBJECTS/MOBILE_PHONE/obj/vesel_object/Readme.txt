--------------- DESCRIPTION ------------------------------------------------
____________________________________________________________________________

- Collection of 16 Stylish Vases.

- Medium resolution that combines quality and performance (Feel free to add as mush resolution as 
you want, the shape will hold itself perfectly.)

- UVs template files integrated, and each vase has its own template, so you can easily design your own vases.  


____________________________________________________________________________
--------------- System requirements-----------------------------------------
____________________________________________________________________________

This product was created in Maya 2014 on a PC. It was not tested on other Version of Maya or on a MAC system.

The main version (.ma and .mb files) comes with V-Ray, mental ray and standard materials. 

Fbx, Obj and Max formats contain standard materials.
